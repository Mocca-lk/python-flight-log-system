# Aviation Maintenance Log
 - Scripts em Python e Queries SQL pensando em problemas  relacionados com aviação e logística.
## O objetivo desse script é a criação de registros de manutenção das aeronaves de uma companhia aérea.
- As informações registradas são o modelo por prefixo, no caso escolhi o prefixo AER-NúmeroAeronave e a manutenção que foi realizada. Mas isso poderia ser alterado por outras necessidades de registor, como compras, registro de problemas apresentados. A opção 1 é para adicionar o prefixo da aeronave. Opção 2 é para apresentar o histórico já registrado e a últiuma opção é para sair do programa.
